# 📡 The Echo Protocol

## 🧠 About

The Echo Protocol is a choice-based, narrative-driven mystery game built with React.  
Players explore a fragmented investigation into a series of unexplained disappearances where every decision shapes the direction of the story.

Over time, missing persons reported similar experiences before vanishing:
- Hearing a voice calling them
- Experiencing déjà vu
- Seeing places no one else could perceive

No clear explanation exists, and the deeper the investigation goes, the less stable the truth becomes.

---

## 🎮 Gameplay

- Branching narrative choices
- Multiple story paths
- Consequence-based progression
- No single correct ending

Every choice may reveal new information or permanently close other paths.

---

## 🧩 Features

- JSON-based story system
- AI-generated narrative structure
- Dynamic branching logic
- React-based UI
- Browser-based experience
